# Materialize Laravel App
**originated by : Malik H**

Web Aplikasi Crud di lengkapi dengan Login, Registrasi Laravel,Materialize CSS,dan untuk graph menggunakan HighChart : 
1. ubah settingan database : 
- di config/database
- jalankan perintah : 

> cp .env.example .env

- lalu ubah .env nya
2. **jalankan perintah:** 

> php artisan migrate

3. **run app dengan :**

> php artisan serve
